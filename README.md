# KWEB_HomePage
* KWEB HomePage (Non-Official)

### Description
---
2019년 1학기 KWEB 보충스터디 실습 참고자료입니다.
간단한 KWEB 홈페이지를 만드는 것이 목표입니다.
전체 홈페이지는 랜딩 페이지, 커리큘럼 소개, 네트워킹, 방명록으로 이루어져 있습니다.
